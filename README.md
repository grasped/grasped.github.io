## Installation
```
pip install mkdocs
mkdocs serve
```

## Usage
All write-ups can be found in `docs`. Everything is written in Markdown format.

## Deployment
This script builds and deploys the docs to grasped.github.io
```
mkdocs gh-deploy
```
